## 创建用户：填写身份信息

```
CREATE USER '用户名'@'主机名' IDENTIFIED BY '密码';
```

- **'用户名'**：你自己起的登录名，如 `'project_user'`。
    
- **'主机名'**：允许谁登录。
    
    - 填 `'localhost'`：只允许在本机登录。
        
    - 填 `'%'`：允许从任何电脑远程登录。
        
- **'密码'**：登录口令，如 `'MyPassword123!'`。
    

---

## 授权：填写权限范围

```
GRANT 权限列表 ON 数据库名.表名 TO '用户名'@'主机名';
```

- **权限列表**：你想让他干什么？
    
    - 只读：填 `SELECT`
        
    - 增删改查：填 `SELECT, INSERT, UPDATE, DELETE`
        
    - 全管：填 `ALL PRIVILEGES`
        
- **数据库名.表名**：他能管哪些东西？
    
    - 某张表：如 `mydb.teacher`
        
    - 某个库的所有表：如 `mydb.*`
        
    - 全服务器：如 `*.*`
        
- **'用户名'@'主机名'**：填你在第一步创建好的那个完整 ID。
    

---

## 撤回：填写收回的内容

```
REVOKE 权限列表 ON 数据库名.表名 FROM '用户名'@'主机名';
```

- **注意**：这里的 **权限列表** 和 **范围** 必须是你之前 `GRANT` 过的。如果你没给过他 `DELETE` 权限，现在去 `REVOKE DELETE` 是会报错的。```
