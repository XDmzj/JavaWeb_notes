# 题目

现在运营想要了解**浙江大学的用户**在不同难度题目下答题的正确率情况 ，请取出相应数据，并按照准确率升序输出。

|3|6543|female|20|北京大学|3.2|12|3|30|

示例： question_practice_detail

|1|2138|111|wrong|

|6|2315|116|right|

示例： question_detail

|   |   |
|---|---|
|question_id|difficult_level|
|111|hard|
|112|medium|
|113|easy|
|115|easy|
|116|medium|
|117|easy|


根据示例，你的查询应返回以下结果：

|   |   |
|---|---|
|difficult_level|correct_rate|
|easy|0.5000|
|medium|1.0000|

# 题解

```sql
select difficult_level,
    sum(
        case
        when qp.result='right' then 1
        else 0
        end
    )
    /count(*)
as
correct_rate
from question_practice_detail qp
left join  user_profile up on up.device_id = qp.device_id
left join question_detail qd on qd.question_id = qp.question_id
where university = '浙江大学'
group by difficult_level
order by correct_rate
;
```
