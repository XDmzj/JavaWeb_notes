题目：现在运营想要将用户划分为25岁以下和25岁及以上两个年龄段，分别查看这两个年龄段用户数量

**本题注意：age为null 也记为 25岁以下**

示例：user_profile

|   |   |   |   |   |   |   |   |   |
|---|---|---|---|---|---|---|---|---|
|id|device_id|gender|age|university|gpa|active_days_within_30|question_cnt|answer_cnt|
|1|2138|male|21|北京大学|3.4|7|2|12|
|2|3214|male||复旦大学|4|15|5|25|
|3|6543|female|20|北京大学|3.2|12|3|30|
|4|2315|female|23|浙江大学|3.6|5|1|2|
|5|5432|male|25|山东大学|3.8|20|15|70|
|6|2131|male|28|山东大学|3.3|15|7|13|
|7|4321|male|26|复旦大学|3.6|9|6|52|

  

根据示例，你的查询应返回以下结果：

|   |   |
|---|---|
|age_cut|number|
|25岁以下|4|
|25岁及以上|3|
# 题解
~~~sql
select
	case
		when age < 25 or age is null then '25岁以下'
		else '25岁及以上'
		end as age_cut
	,
	count(*) as number
from user_profile
group by age_cut
order by
	case
	    when age_cut = '25岁以下' then 1
	    else 3
	end;
~~~


## case的用法
### CASE 语法位置详解

在不同的场景下，`CASE` 各个位置应该写什么：

####  在 `SELECT` 中（转换值）

- **WHEN**：写判断条件（如 `score > 60`）。
    
- **THEN**：写你想显示的结果（如 `'及格'`）。
    
- **ELSE**：写如果不满足上述所有条件时的默认值（如 `'不及格'`）。
    
- **END AS**：写这一列的标题名。
    

#### 在 `GROUP BY` 中（自定义分类）

- **用途**：当你不想按原始字段分组，而是想按“逻辑范围”分组时使用。
    
- **写法**：通常直接写整个 `CASE` 表达式，或者直接引用 `SELECT` 中定义的别名（取决于数据库版本）。
    

#### 在 `ORDER BY` 中（自定义排序）

- **用途**：当升序（ASC）或降序（DESC）无法满足需求时。
    
- **例子**：比如你想让“北京”排第一，“上海”排第二，其他随便排。
    
- **写法**：`THEN` 后面通常跟**数字**（1, 2, 3...），数字越小排在越前面。



### case进行多次判断
#### 多次判断语法

SQL 会**从上到下**依次匹配，一旦满足某个 `WHEN` 的条件，就会执行对应的 `THEN` 并立即结束这个 `CASE`。

```
SELECT 
    device_id, 
    gender, 
    CASE 
        WHEN age < 20 THEN '20岁以下'      -- 第一次判断
        WHEN age BETWEEN 20 AND 24 THEN '20-24岁'  -- 第二次判断
        WHEN age >= 25 THEN '25岁及以上'   -- 第三次判断
        ELSE '其他'                        -- 兜底处理（如 age 为 NULL）
    END AS age_cut
FROM user_profile;
```

---

#### 前面的when判断成功，后面的when就不执行了

在刷牛客或 LeetCode 时，一定要注意条件的**包含关系**。因为 `CASE` 是“短路式”执行的：

- **错误示例：**
```
    CASE 
        WHEN age < 30 THEN '30岁以下'
        WHEN age < 20 THEN '20岁以下' -- 这一行永远不会被执行！因为 <20 肯定满足 <30
    END
```
    
- **正确做法：** 应该**从范围小的开始写**，或者使用明确的区间（如 `BETWEEN`）。
    

---

#### 多条件组合判断

如果你想在一次判断中同时考虑多个字段（比如年龄和性别），可以使用 `AND` 或 `OR`：

```
CASE 
    WHEN age < 25 AND gender = 'male' THEN '年轻男性'
    WHEN age < 25 AND gender = 'female' THEN '年轻女性'
    WHEN age IS NULL THEN '信息缺失'
    ELSE '其他群体'
END AS user_group
```
