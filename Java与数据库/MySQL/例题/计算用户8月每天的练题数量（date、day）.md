**示例：question_practice_detail**

|   |   |   |   |   |

根据示例，你的查询应返回以下结果：

|     |              |
| --- | ------------ |
| day | question_cnt |
| 13  | 5            |
| 14  | 2            |
| 15  | 3            |
| 16  | 1            |
| 18  | 1            |

# 题解


```sql
select day(date) as day ,
count(id) as question_cnt
from question_practice_detail
where month(date) = 8
group by date
```




## 日期相关知识

### DATE：数据类型

`DATE` 通常作为字段的类型，专门用来存储“年-月-日”信息。

- **格式**：`YYYY-MM-DD`（例如：`2026-04-08`）。
    
- **特点**：它只包含日期，**不包含**具体的小时、分钟或秒（那是 `DATETIME` 或 `TIMESTAMP` 干的事）。
    
- **场景**：存储生日、入职日期、订单日期。
    

```sql
-- 创建表时定义
CREATE TABLE orders (
    id INT,
    order_date DATE
);
```




### day\month\year

假设我们有一个日期变量 `@dt = '2026-04-08 18:30:00'`：

|**函数**|**作用**|**结果示例**|**结果范围**|
|---|---|---|---|
|**`YEAR()`**|提取**年份**|`2026`|1000 ~ 9999|
|**`MONTH()`**|提取**月份**|`4`|1 ~ 12|
|**`DAY()`**|提取**天数**|`8`|1 ~ 31|
